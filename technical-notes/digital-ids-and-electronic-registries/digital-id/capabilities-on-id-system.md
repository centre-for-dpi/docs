# Capabilities on ID system

Building a use cases layer on top of any foundational / functional ID system is crucial for unlocking service delivery. An ID system should be considered infrastructure that provides citizens with seamless access to services.

An ID system is only as powerful as the use cases it can unlock. This is also how countries can encourage enrollment and ensure that their ID system is “sticky”.

The layers built on an identity system, as outlined below, can make an existing ID system operate as digital public infrastructure in an economy, thereby enhancing both enrollment rates and overall system utilization.
